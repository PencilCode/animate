`sin x` computes the sine of the angle `x`, where `x` is measured
in degrees:

<pre class="jumbo">
write sin 90
</pre>

To compute the sine in radians, you can use `Math.sin` instead:

<pre class="jumbo">
write Math.sin PI/2
</pre>

Other trigonometric functions include
[`cos`](cos.html) and [`tan`](tan.html).
