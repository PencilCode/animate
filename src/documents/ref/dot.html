Dot

Dot draws a dot right under the turtle. It can draw a dot in any color.

dot red

Dot accepts an optional second argument, which is the size of the dot in pixels. Remember that arguments need to be separated by a comma.

dot purple, 100

Coffeescript needs at least two words to run a function, but you can use the word do to run a function without aguments. do dot draws a default black dot.

do dot
