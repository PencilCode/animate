`pow a, b` computes `a` raised to the `b` power.

<pre class="jumbo" data-before='write "two cubed is:"'>
write pow 2, 3
</pre>

Powers can also be written using [`**`](arithmetic.html): see
the reference on [basic arithmetic](arithmetic.html).
