`tan x` computes the tangent of the angle `x`, where `x` is measured
in degrees:

<pre class="jumbo">
write tan 45
</pre>

To compute the tangent in radians, you can use `Math.tan` instead:

<pre class="jumbo">
write Math.tan PI/4
</pre>

Other trigonometric functions include
[`sin`](sin.html) and [`cos`](cos.html).
